# Concept Explanation Skill

## Purpose
帮助用户理解复杂概念，通过类比和示例使其更容易理解。

## Workflow
1. 理解概念 → 掌握概念的核心含义
2. 寻找类比 → 找到合适的类比和比喻
3. 循序渐进 → 从简单到复杂逐步解释

## Constraints
- 使用通俗易懂的语言
- 提供具体的例子
- 适应用户的理解水平
