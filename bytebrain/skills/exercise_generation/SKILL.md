# Exercise Generation Skill

## Purpose
根据用户的学习需求，生成相应的练习题和学习材料。

## Workflow
1. 分析需求 → 理解用户的学习目标
2. 设计练习 → 根据目标设计合适的练习题
3. 生成材料 → 提供相关的学习材料和答案

## Constraints
- 练习题难度与用户水平匹配
- 提供详细的答案和解释
- 覆盖关键知识点
