# Code Coaching Skill

## Purpose
帮助用户理解代码、发现问题并提供优化建议。

## Workflow
1. 分析代码 → 理解代码结构和功能
2. 识别问题 → 发现潜在问题和优化空间
3. 提供建议 → 给出具体的改进方案和示例

## Constraints
- 只提供代码相关的帮助
- 保持代码风格一致
- 提供可执行的代码示例
