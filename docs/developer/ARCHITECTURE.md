# 架构设计

---

## 系统架构

ByteBrain 采用模块化设计，分为以下几个层次：

```
┌─────────────────────────────────────────────────────────────┐
│                      UI 层 (Streamlit)                       │
└──────────────────────────────┬──────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│                    Agent 编排层 (LangGraph)                  │
└──────────────────────────────┬──────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│                    Skill 层 (模块化能力)                     │
└──────────────────────────────┬──────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│              核心层 (LLM / Embedding / RAG)                 │
└──────────────────────────────┬──────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│                   数据层 (向量数据库 / 文件)                  │
└─────────────────────────────────────────────────────────────┘
```

---

## 核心模块

### 1. Core Layer (核心层)
- `llm.py` - 大语言模型封装
- `embedding.py` - 向量模型封装
- `vector_store.py` - 向量存储
- `rag.py` - RAG 系统
- `agent.py` - Agent 框架

### 2. Skills Layer (技能层)
- `base.py` - Skill 基类
- `knowledge_retrieval.py` - 知识检索
- `code_coaching.py` - 代码教练
- `concept_explanation.py` - 概念解释

### 3. Guardrails Layer (防护层)
- `input.py` - 输入防护
- `output.py` - 输出防护
- `rules.py` - 规则引擎

### 4. Prompts Layer (提示层)
- `system.py` - 系统提示
- `templates.py` - 模板库
- `few_shot.py` - Few-Shot 示例

---

## 数据流程

```
用户输入
    ↓
Input Guardrails
    ↓
Context Engineering
    ↓
Agent Orchestration
    ↓
Skill Execution
    ↓
Output Guardrails
    ↓
用户输出
```
