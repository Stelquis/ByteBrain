## 基础使用

### 启动应用
```bash
make run
```

打开浏览器访问 `http://localhost:8501`

---

## 上传笔记

### 支持的格式
- Markdown (.md)
- 文本文件 (.txt)

### 上传步骤
1. 点击侧边栏的"上传笔记"
2. 选择文件或文件夹
3. 等待向量化完成
4. 开始使用！

---

## 智能问答

### 提问技巧
- 清晰具体地描述问题
- 提供上下文（如相关概念）
- 说明你期望的解释深度（入门/进阶/专家）
