# 快速开始

---

## 前置要求

- **Python 3.11+**（推荐 3.12）
- 8GB+ RAM（推荐 16GB）

> **为什么选择 Python 3.11+？**
> - Python 3.9 已于 2025-10-31 结束生命周期
> - Python 3.11+ 性能提升约 25%
> - 更好的类型提示支持
> - 企业主流版本，生态成熟

---

## 安装

```bash
git clone https://github.com/your-username/ByteBrain.git
cd ByteBrain
pip install -e .
```

---

## 运行

```bash
make run
```

打开浏览器访问 `http://localhost:8501`
