# 快速开始

---

## 前置要求

- **Python 3.12+**（2026 年新项目首选）
- 8GB+ RAM（推荐 16GB）

> **为什么选择 Python 3.12？**
> - Python 3.9 已于 2025-10-31 结束生命周期
> - Python 3.12 是 2026 年新项目的首选版本
> - 性能优异，新特性多，生态基本成熟
> - 支持到 2028-10，长期稳定

---

## 安装

```bash
git clone https://github.com/your-username/ByteBrain.git
cd ByteBrain
pip install -e .
```

---

## 运行

```bash
make run
```

打开浏览器访问 `http://localhost:8501`
